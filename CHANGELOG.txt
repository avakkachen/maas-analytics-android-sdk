MaaSAnalytics Changelog
==========

Version 1.3.4 *(2014-02-26)*
----------------------------
 * Fixed builds to produce Java 6 compatible binaries using 'sourceCompatibility' and 'targetCompatibility' equal to '1.6'.
 * Requires MaaSCore v1.3.5
 
Version 1.3.3 *(2013-11-08)*
----------------------------
 * Requires Core v1.3.2
 * Removed alpha numeric checks on event names and event parameters.
 * Minor optimizations.

Version 1.3.1 *(2013-08-06)*
---------------------------- 
 * Requires MaaSCore 1.2.1
 * Fixed issues with some analytics not sending when parameters are added.

Version 1.3.0 *(2013-08-06)*
----------------------------
 * Event Names and Parameters must now be alphanumeric strings.

Version 1.2.0 *(2013-07-31)*
----------------------------
 * Requires MaaSCore 1.2.0
 
Version 1.1.1 *(2013-07-30)*
----------------------------
 * Minor bug fixes
